public subroutine subblock(quadblock, factors, positions, curr_weight, spline1, spline2)

Builds the quadblock. The four subblocks are calculated based on the specified factors, positions in the block, weights and basis functions. These four subblocks are then added to the quadblock that is passed along, and the filled quadblock is returned.

public subroutine reset_factors(factors, size_factors)

Resets the factors array. If factors is allocated, it is deallocated and reallocated with size size_factors. Afterwards it is initialised to zero.

public subroutine reset_positions(positions, size_positions)

Resets the positions array. If positions is allocated, it is deallocated and reallocated using size_positions. Afterwards it is initialised to zero. The new size is always (size_positions, 2) .
